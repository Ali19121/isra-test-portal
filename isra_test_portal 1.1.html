LAN Setup Guide

How to run this portal on a school network so 50+ candidates can take tests simultaneously from different computers.

Option 1 — Simplest: Shared Folder on School Network
No installation needed. Works with Windows shared folders.
  1. Copy isra_test_portal.html to a shared folder on the main/admin computer
  2. On every candidate computer open File Explorer → go to the shared folder
  3. Double-click the HTML file — it opens in browser
  4. Each candidate logs in with their name, CNIC and test password
  5. Results are saved locally on each computer and sent to Google Sheet (if internet available)
Note: In this method each computer stores its own history. To collect all results, check Google Sheet (recommended) or collect the Excel exports from each computer.
Option 2 — Best: Python Local Web Server (Recommended for 50+ candidates)
All candidates share one central data source. Requires Python (free, already installed on most school computers).
Step 1: On the ADMIN computer, open Command Prompt (CMD) and run:
cd Desktop
python -m http.server 8080
Step 2: Find the admin computer's IP address — open CMD and type:
ipconfig
Look for IPv4 Address — it will be something like 192.168.1.5
Step 3: On every candidate computer, open browser and go to:
http://192.168.1.5:8080/isra_test_portal.html
This works for 50+ simultaneous candidates — each candidate accesses the same file from the server. Results go to Google Sheet automatically if internet is available. No installation on candidate computers needed.
For IT Staff — Important Notes
Technical details to ensure smooth operation
  • The portal is a single HTML file — no database, no server software needed
  • Each browser tab is completely independent — 50 candidates = 50 separate sessions
  • Questions are stored in localStorage — if using shared server method, admin must first open the portal on server computer and upload questions. Candidates on other computers will see the same HTML file but will each have their own localStorage
  • Recommended browser: Google Chrome or Microsoft Edge (latest version)
  • Disable browser auto-complete on candidate computers to prevent data leakage
  • For best performance: ensure school WiFi/LAN is working before test starts
  • Results backup: always configure Google Sheet so results are saved centrally
  • If a candidate's browser crashes mid-test, their progress is lost — advise them not to close the browser

🔗 Google Sheet Integration

Paste your Google Apps Script Web App URL below. Every candidate result will be automatically saved to your sheet with all details including AI analysis of descriptive answers.

Paste this code in Apps Script:
function doPost(e) { var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName("Results") || ss.getActiveSheet(); var d = JSON.parse(e.postData.contents); if(sheet.getLastRow()===0){ sheet.appendRow(["Date","Time","Name","Father Name", "CNIC","Test","Score%","Correct","Wrong", "Skipped","Total MCQ","Time Taken","AI Analysis"]); } sheet.appendRow([d.date,d.time,d.name,d.father, d.cnic,d.test,d.score+"%",d.correct,d.wrong, d.skipped,d.totalMcq,d.timeTaken,d.aiAnalysis||""]); return ContentService .createTextOutput(JSON.stringify({status:"ok"})) .setMimeType(ContentService.MimeType.JSON); }
  1. Open Google Sheets → Extensions → Apps Script
  2. Paste the code above, save project (Ctrl+S)
  3. Deploy → New Deployment → Web App
  4. Set "Who has access" to Anyone → Deploy
  5. Copy Web App URL → paste above → Save
